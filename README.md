# OS Bakery Simulator

**Temat 15** — Systemy Operacyjne

Symulacja ciastkarni z samoobslugowym sklepem firmowym.
Wieloprocesowy system oparty na IPC System V (pamiec dzielona, semafory, kolejki komunikatow) oraz lacza (pipe, FIFO).

## Struktura

- `src/` — kod zrodlowy
- `docs/` — dokumentacja
- `tests/` — testy integracyjne
